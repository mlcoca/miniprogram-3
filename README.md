# 微信便签小程序

一个简约风格的笔记便签小程序，用于快速记录和管理日常笔记。

## 功能特点
- 支持创建、编辑、删除笔记
- 简洁的列表视图
- 支持笔记标题和内容
- 按创建时间排序
- 笔记搜索功能
- 笔记分类管理
- 笔记分享功能
- 数据云端备份

## 页面结构
- pages/index/index - 笔记列表页面
- pages/edit/edit - 笔记编辑页面
- pages/category/category - 分类管理页面
- pages/backup/backup - 数据备份页面

## UI设计风格
- 主色调: #07C160 (微信绿)
- 背景色: #F7F7F7
- 文字颜色: #333333
- 次要文字: #999999
